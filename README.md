# Stock Analysis Project

This project aims to provide a comprehensive analysis of stocks using various data sources and scoring models.

## Directory Structure

- `app/`: Contains the application code.
- `systemd/`: Contains the systemd service and timer files.
- `config/`: Contains configuration files.
- `requirements.txt`: Lists Python dependencies.
- `README.md`: Project documentation.
- `.gitignore`: Specifies files to ignore in the repository.
- `setup.py`: Provides packaging information.

## Installation

1. Clone the repository.
2. Install dependencies listed in `requirements.txt`.
3. Configure the application using `config/config_template.yaml`.
4. Setup systemd service and timer for automation.

## Usage

Run the main application using:
```
python app/main.py
```
